Audio

До HTML5 не было стандарта для воспроизведения аудио файлов на веб-страниц.
Элемент <audio> HTML5 определяет стандарт для встраивания аудио в веб-страницу.


Есть два разных способа указать URL файла аудио источника. Первый использует атрибут источника:


        <audio src="http://d.zaix.ru/jI4B.mp3"></audio>


Второй способ использует элемент <source> внутри элемента <audio>:


        <audio>
            <source src="audio.mp3" type="audio/mpeg">
            <source src="audio.ogg" type="audio/ogg">
        </audio>


Элемент <audio> создает аудио плеер внутри браузера. Вот что получается:


Если же браузер не поддерживает данный тег, то внутри тега audio стоит написать текст, который будет отображаться в случае ошибки.


Важно!

Вам может не понадобиться некоторые из этих элементов, в зависимости от структуры вашей страницы.
