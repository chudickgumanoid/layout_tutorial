Видео

Элемент видео похож на элемент аудио.
Вы можете указать URL источника видео, используя атрибут в элементе video или используя элементы source внутри элемента video:


        <video controls>
            <source src = "video.mp4" type = "video / mp4"> 
            <source src = "video.ogg" type = "video / ogg">
            Видео не поддерживается вашим браузером
         </video>



Важно!

Другой аспект, который объединяет аудио и видео элементы, заключается в том, что основные браузеры не поддерживают одинаковые типы файлов. Если браузер не поддерживает первый тип видео, он попробует следующий.
