Progress

Элемент <progress> предоставляет возможность создавать индикаторы выполнения в сети.
Элемент прогресса может использоваться внутри заголовков, абзацев или в любом другом месте тела. Значение атрибутов элемента прогресса: указывает, сколько процентов было выполнено.

Первый способ вставить svg-изображение это через тег img


    <progress min="0" max="100" value="35"> </progress>



Важно!

Используйте тег <progress> в сочетании с JavaScript для динамического отображения хода выполнения задачи.
