Формы


Формы в HTML используются для сбора информации от пользователя. Формы создаются с помощью элемента form, который имеет открывающий и закрывающий теги:


<html>
<head>
    <title>Формы</title>
</head>
<body>
        <form>
            ...
        <form>
</body>
</html>


Атрибут action указывает на веб-страницу, которая должна загрузиться после отправки формы пользователем.

Атрибуты method и name

Атрибут метод определяет метод HTTP (GET или POST) , которые будут использоваться, когда формы представлены:

<form action="url" method="GET">




 <form action="url" method="POST">


Чтобы принять пользовательский ввод, вам нужны соответствующие элементы формы, такие как текстовые поля. Элемент <input> имеет много вариантов, в зависимости от атрибута type. Это может быть текст, пароль, радио, URL, отправить и т. Д.



Элементы форм

Если мы изменим тип ввода на радио, это позволит пользователю выбрать только один из нескольких вариантов:


    <input type = " radio " name = "пол" value = " мужчина " /> мужчина <br/> 
    <input type = " radio " name = "пол" значение = " женщина " /> женщина <br/>





    <input type = " submit " value = "Submit" />


Важно!

Формы отправляются на веб-страницу, которая расположена на сервере.


Атрибут name указывает имя для формы.


Тег input не имеет закрывающего тега.
