Списки

Нумерованные списки

Нумерованный список начинается с тега <ol>, а каждый пункт указывается с помощью тега <li>.
Внизу пример нумерованного списка


<html>
<head>
    <title>Списки</title>
</head>
<body>
        <ol>
            <li>Первая строка</li>
            <li>Вторая строка</li>
            <li>Третья строка</li>
            <li>Лев</li>
        <ol>
</body>
</html>

Маркированный список

Тут всё то же самое, но вместо цифр указываются маркера.



<html>
<head>
    <title>Списки</title>
</head>
<body>
        <ul>
            <li>Первая строка</li>
            <li>Вторая строка</li>
            <li>Третья строка</li>
            <li>Лев</li>
        <ul>
</body>
</html>


Результат:




Важно!

Для нумерованного списка все пункты будут автоматически пронумерованы.


Каждый пункт списка всегда идет с отдельным маркером.
