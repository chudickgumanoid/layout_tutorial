Структура HTML страницы

Если открыть любую веб-страницу, то она будет содержать в себе типичные элементы, которые не меняются от вида и направленности сайта.


Самая простая структура страницы выглядит вот так:


<!DOCTYPE html>
<html lang="en">
<head>
　 <meta charset="UTF-8">
　 <title>Document</title>
</head>
<body>
　 <h1>Lorem ipsum dolor sit amet.</h1>
　 <p>Lorem ipsum dolor sit amet, consectetur adipisicing elit.
　 　 Ratione, soluta.</p>
</body>
</html>

Результат данного кода:



Тег <html> </html>

Между этими тегами заключается всё содержимое html страницы.


<head> </head>

За открывающим тегом html следует заголовок документа, который определяется открывающими и закрывающими тегами head.


Данный заголовок содержит все не визуальные элементы страницы, которые управляют этой страницей. То есть в теге <head> </head> находится название вашей страницы, подключенные стили css и js, так же шрифты и параметры meta-данных.



<body> </body>

После тега заголовка head следует тег тела страницы - body. В теге body заключаются все элементы, отвечающие за визуальное оформление документа.


Заголовки, текста, параграфы, ссылки, изображения, блоки и многое другое всего лишь часть того, что может быть размещено в теге body.


В итоге получается вот такая структура HTML документа:

<html>
<head>
</head>
　<body>
　</body>
</html>
